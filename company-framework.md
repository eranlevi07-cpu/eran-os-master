## 1. ERAN PRO

### מה זה
מותג פרימיום חדש. כל הפרסום מוביל לכאן. רושם ראשוני של הפקות רציניות ויקרות.

### שירותים
- נופשים פרימיום (בארץ + חול)
- Incentive / ימי מנהלים
- הפקות ערב חברה
- XSTREAM BAR כ-add-on בלעדי

### קהל יעד
- מנהלות רווחה בחברות גדולות
- מפיקים אחרים ללקוחות שלהם
- תקציבים גבוהים

### זהות ויזואלית
- טאגליין רשמי: **"HIGH-END PRODUCTION"**
- צבעים: Navy (#1B2A4A) + אפור כסף
- לוגו: EP מונוגרם עם אלמנט חיבור טכנולוגי

### סטטוס דיגיטלי
- דומיין: eran-pro.co.il
- לוגו: קיים (מרץ 2026)
- אתר: **צריך לבנות**

### מסרים שיווקיים
- הפקות ברמה אחרת
- ניסיון של 20+ שנה
- מוצרים בלעדיים (XSTREAM BAR)

## 7. EVENT-IQ (לשעבר YOMKEF OS / TRIPIQ)

### מה זה
פלטפורמת B2B SaaS לניהול אירועי רווחה ארגוניים מקצה לקצה.
מחברת בין 4 שחקנים: מנהלת HR, מפיק, ספקי שטח, עובדים.

### שותפים
- ערן - יזם + ידע תחום
- אדם - מנהל מוצר
- רוני - מתכנת

### 4 פורטלים
1. **HR** - דשבורד שנתי, בונה RSVP, מרקטפלייס ספקים, דוח ROI
2. **מפיק** - P&L, תזרים, לוגיסטיקה, פייפליין לידים, Plan B
3. **ספקים** - לוח בקרה, ניטור ביטוח, WhatsApp אוטומציה
4. **עובד (אפליקציה)** - RSVP, צ'ק-אין, גלריה חיה, משחקים, משוב

### טכנולוגיה
- React + TypeScript + Tailwind CSS
- ~40 מסכים
- נבנה ב-Lovable כרגע, יועבר ל-Claude Code בהמשך
- RTL מלא, White-Label, AI מובנה, Real-Time, Gamification

### מודל הכנסות
| קהל | מחיר חודשי |
|---|---|
| מפיקים | 300 - 990 ₪ |
| ספקים | 100 - 300 ₪ |
| HR (ריטיינר) | 700 ₪ + % מעסקאות עצמאיות |

### סטטוס
- בפיתוח פעיל ב-Lovable
- PRD מפורט קיים (שיחות Gemini)
- מעבר ל-Claude Code מתוכנן
